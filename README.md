# 术图 Shutu Analysis

面向中文学术场景的数据可视化分析助手。

📊 上传一份数据 → 🤖 AI 生成三套创新图表方案 → 💡 通俗解读 → 📎 一键导出报告

## 核心亮点

- **双模型分工**：ECNU 负责数据概览与通俗解读，DeepSeek 负责高质量方案生成  
- **三套方案对比**：每种分析都提供三种互补视角，降低选择门槛  
- **零门槛可用**：普通模式下无需任何 API Key 也能完成完整分析流程  
- **可编辑可复现**：所有图表均可在线编辑 JSON，Python 代码一键复制  
- **中文排版预设**：内置中文字体、配色、DPI 等学术参数  
- **多维结果输出**：支持 Markdown、长图 PNG、A4 PDF、只读分享链接
